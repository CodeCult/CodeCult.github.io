Programmeren Workshop: Girls Day

Wist jij dit? De allereerste programmeur was een vrouw. Haar naam was Ada Lovelace en zij was de dochter van de beroemde dichter Lord Byron. Haar moeder was bang dat kleine Ada teveel op haar vader ging lijken en zij dwong Ada om wiskunde te studeren. Als tiener werd zij gevraagd om een paper van Charles Babbage te vertalen naar het Engels. In deze paper werd een eerste prototype van een computer beschreven en zij schreef voor deze (nog niet bestaande) computer een programma om Bernoulli getallen te berekenen. Zij was dus niet alleen de eerste programmeur, zij was dit ook nog eens voordat computers bestonden!

Tijd om te beginnen

Ga allereerst naar de python sandbox en klik op de Python knop in het midden. Links kan je code gaan schrijven en als je dan op de Run knop drukt, dan wordt jouw code rechts uitgevoerd. Volg de onderstaande opdrachten en neem de eerste stappen om de nieuwe Ada Lovelace te worden.

Opdracht 1: Hello World

Het allereerste dat je moet doen als je een programmeer-taal gaat leren is een Hello World programma te schrijven. Zo'n programma doet niks anders dan hallo zeggen tegen de persoon die het programma uitvoert. In Python is dit snel gemaakt, probeer maar het volgende uit te voeren: 

print "Hello World"

Opdracht 2: Rekenen

Laten we nu proberen iets leukers te printen. Computers zijn erg goed in rekenen en Python werkt goed als rekenmachine. Je kan getallen optellen, aftrekken, vermenigvuldigen en delen, maar ook machten rekent hij zo uit. 

print 3+14

Bij die laatste is iets vreemds aan de hand. Hoe zou dit kunnen komen?

Opdracht 3: Tot 10 tellen

Laten we nu proberen iets leukers te printen. Computers zijn erg goed in rekenen en het belangrijkste van rekenen is kunnen tellen. Bij programmeren doen wij dit met een for-loop 

for i in range(10):
  print i

Als het goed is zie je dat computers ergens ander beginnen met tellen dan wat mensen doen. Kan jij nu hiermee de tafel van 4 uitprinten? Tip hierbij! Met print 5, print hij op 1 regel. Dus door een komma achter je print te zetten. Je kan ook bijvoorbeeld print 5, "+", 6, 5+6 gebruiken om een complexe regel te printen. Als je wilt dat de computer bij 1 begint en tot en met 10 doortelt dan kan je het volgende doen:

for i in range(1,11):
  print i

Opdracht 5: Als dit, dan dat

Soms heb je een stukje code die alleen hoeft uigevoerd te worden als iets waar is. Bijvoorbeeld: Stel je wilt weten of je naar een feest mag. Dit is afhankelijk van tot hoe laat het feest is en hoe oud jij bent 

leeftijd = 14
feestDuurtTot = 22
if leeftijd>=16 or feestDuurtTot<22:
  print "Jij mag naar het feest"
else:
  print "Jij blijft gezellig thuis"

Na zo'n if staan een stel condities. Als alle condities waar zijn dan voert het programma de code uit. Wat na else staat wordt alleen uitgevoerd als de condities niet waar waren. Je kan op verschillende manier controleren. 

leeftijd = 17
feestDuurtTot = 22
if (not leeftijd > 16) and (feestDuurtTot >= 22):
  print "Jij blijft gezellig thuis"
else:
  print "Jij mag naar het feest"

Maak voor iedere persoon in jouw groepje een variabele waar je in zet hoeveel geld je hebt. Schrijf een programma dat kijkt of jullie samen genoeg geld hebben om naar de bioscoop te gaan.

Opdracht 6: Functioneel

Het begint nu misschien al duidelijk te worden dat het vervelend is om iedere keer iets in je programma aan te passen en dan opnieuw uit te voeren. Als je een stukje code vaak wilt uitvoeren met verschillende waardes, dan is het verstandig om een functie te maken. 

def colaDrinken(naam, tijd, leeftijd, geldInZak):
    if tijd>12 and leeftijd>=14 and geldInZak>=1.50:
        print naam, "kan en mag cola drinken"
    else:
        if geldInZak<1.50:
            print naam, "heeft te weinig geld"
        elif leeftijd<14:
            print naam, "is niet oud genoeg"
        elif tijd<=12:
            print "Het is te vroeg voor cola voor", naam

colaDrinken("Robrecht", 13, 26, 1)
colaDrinken("Sandra", 15, 15, 2)
colaDrinken("Roos", 11, 12, 4)
colaDrinken("Sofia", 11, 18, 3)
colaDrinken("Eliza", 13, 12, 1)

Maak een functie die gegeven uursalaris en gewerkte uren berekent hoeveel je hebt verdiend.

Opdracht 7: pi

pi is een zeer handig getal. Wij kunnen pi makkelijk gebruiken met de volgende code: 

from math import pi

print "PI =", pi
def omtrekEnOppervlakVanCirkel(radius):
	print "cirkel met radius van", radius, "cm:\t omtrek =", pi*(radius+radius), "en oppervlak =", pi*radius*radius

omtrekEnOppervlakVanCirkel(1)
omtrekEnOppervlakVanCirkel(5)
omtrekEnOppervlakVanCirkel(15)
omtrekEnOppervlakVanCirkel(25)

Het enige wat jammer is, is dat pi hier niet zoveel decimalen heeft. Daarom gaan we pi nu zelf berekenen! De formule om dat te doen is:
pi = 4/1 - 4/3 + 4/5 - 4/7 + 4/9 - 4/11 + 4/13 - ....
en dat gaat dan voor eeuwig door. Een computer kan dit snel uitrekenen en daarom gaan wij nu een functie maken die dit voor ons gaat doen. Vul de rest van de functie aan en kijk of je pi kan berekenen 

def berekenPi():
    # Dit is het begin punt
    pi = 4
    for i in range(1000):
        # Wat moeten wij hier doen?
        pi = pi
    return pi
        
pi = berekenPi()
print pi

Opdracht 9: Gemiddelde cijfer

Al die wiskunde is leuk, maar wat kan je er nou eigenlijk mee. Veel rekenwerk doen is saai en gelukkig zijn computers veel beter en sneller dan wij dat zijn. Stel je wil jouw gemiddelde cijfer berekenen. Hoe pak je dat aan? 

def gemiddeldeCijfer(vak, cijfers, zwaarteVanCijfer):
    # Tel alle cijfers bij elkaar op
    totaal = 0
    for cijfer in cijfers:
        totaal = totaal + cijfer
	# Deel het totaal door het aantal cijfers
	gemiddelde = totaal/float(len(cijfers))
    print "Voor", vak, "sta ik een", gemiddelde
	
gemiddeldeCijfer("Wiskunde", [6,7,9,4,5], [0.2, 0.4, 0.2, 0.3, 0.1])
gemiddeldeCijfer("Engels", [8,10,9,6,7], [0.2, 0.2, 0.1, 0.2, 0.3])

Het bovenstaande programma berekent een gemiddelde voor een lijst met cijfers. Alleen niet ieder cijfer telt even zwaar mee. Kan jij het echte gemiddelde berekenen met de gewichten van de cijfers die in zwaarteVanCijfers staan?
